## SimpleArch is a simplified version of Retroarch for both new and existing users. 

### Features (As of Version 1.0.0)

- Forget combing through directories within the application - SimpleArch includes a Launcher which asks where your games are stored, once clicked the directory is added within SimpleArch
- The best console-like launcher preselected for convenience - XMB set is set as the default launcher after game selection
- Manual Database and Core Info Updates? Not here - SimpleArch updates Database and Core Info using the latest Libretro packages after game selection
- Avoid having to watch tutorials to download cores (More cores planned to be added, please read Future Goals below) - Some cores are pre-installed giving access to SNES, MGBA, SEGA Genesis and Nintendo DS


### Future Goals

- Changing setting names and positions so that what most users need can be found easier
