# SimpleArch

## SimpleArch is a simplified version of Retroarch for both new and existing users. 

### Disclaimer
With Release Version 1.0.0, this project is currently in beta and it is currently only a fork of Retroarch.

### Future Goals
Cores will be included from startup so there is minimal setup
Setting names and positions will be changed so that what most users need can be found easier
A custom logo for SimpleArch branding 
Planned setup which asks where games are stored so that core and game locations are setup from first
execution.
